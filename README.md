Este proyecto representa el entregable del **Nivel Integrador** para el curso de Análisis de Datos en **Talento Tech**. La aplicación permite explorar métricas clave de una cadena de cafeterías, utilizando técnicas avanzadas de visualización y limpieza de datos.

## 🤵 Autor
**Juan Pablo Henao** *Analista de Datos Senior*

## 🚀 Funcionalidades Clave
* **Landing Page Dinámica:** Explicación del contexto del negocio y acceso controlado.
* **Análisis con Seaborn:** Gráficos estadísticos de dispersión y densidad de precios.
* **Visualización Interactiva:** Mapas de calor y áreas de ventas por hora.
* **Documentación Integrada:** Sección de metodología y origen de datos mediante pestañas.

## 🛠️ Requerimientos
Para ejecutar este proyecto, necesitas instalar las dependencias detalladas en el archivo `requirements.txt`:
`pip install streamlit pandas matplotlib seaborn plotly kagglehub`
